# Linkfolio

Personal portfolio for Boluwatife Emmanuel, built with React, Vite, Tailwind CSS, Framer Motion, GSAP, and Lenis.

Most site content is managed through published Google Sheets CSV feeds, so updates can be made without a backend.

## Stack

- React 18
- Vite 6
- Tailwind CSS 3
- Framer Motion
- GSAP
- Lenis
- Google Sheets CSV

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Content

Content is fetched from published Google Sheets CSV sources for:

- profile
- projects
- catalogue
- reviews
- social links

If content changes do not appear immediately, refresh the page and confirm the sheet is still published as CSV.
